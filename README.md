# Plant Seedling Classification using NNDL

## Overview
This project focuses on classifying plant seedlings into different species using deep learning techniques.  
The model analyzes images of young plants and predicts the species automatically. This helps in early plant identification and can support agricultural monitoring.

This project was developed as part of the **Neural Networks and Deep Learning (NNDL)** course.

## Objectives
- To build an image classification model for plant seedlings.
- To apply deep learning techniques for recognizing plant species.
- To improve accuracy in plant identification at an early growth stage.

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib

## Methodology
A Convolutional Neural Network (CNN) is used for image classification.  
The model is trained on a dataset of plant seedling images. During training, the network learns patterns and features from the images to classify them into different plant species.
